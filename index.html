BRICKst

The first Web brick.

BRICKst is yet another mobile-first CSS framework for kickstarting your projects. Inspired by ZURB's Foundation (I mean, I pretty much copied their grid solution), OOCSS, SASS amazing powers and more.

Yes, I know. This "demo" is terrible so far. This will serve a purpose of a demo and documentation at some point in the future. Feel free to skim through dist/css/brickst.scss to discover all the features.

SCSS Mixins

Responsivity

  • @include respond-to(medium/large/hd) { .. }
    Foundation-like breakpoints medium/large + custom-made "HD" break
  • @include respond-from(XX units) { .. }
    Custom breakpoints, when default aren't enough (e.g. respond-from(123px))
  • @include orientation(landscape/portrait) { .. }
    Define media querries for device orientation

Gradients

  • @include gradient($top, $bottom)
    easily create CSS gradients, enter both parameters ("from" and "to" colors); gradient is vertical, top to bottom

HTML

Grid

12-column rows by default, nestable (one level confirmed :)

  • .row – classic row with max-width defined
  • .row.narrow – row with smaller gutters between columns (by half)
  • .row.collapse – no gutters between columns at all
  • .row.expand – full-width row

Buttons

Classic buttons. Default usage via button and/or a.button:

Tiny Default Toggled Small Success Warning Button Large Alert

Optional classes

  • .tiny/.small/.large (or none) – 4 sizes, define via classes
  • .secondary/.success/.alert/.warning (or none) – 5 variants, different coloring by default
  • .toggled – makes button toggle-able (useful for playing with JS)

Panels

Simple panels for whatever you want. This panel is .hide-for-small, therefore only visible on medium and large screens.

Visibility Classes

  • .hide – hide completely (useful for fiddling with jQuery)
  • .hide-for-small, .hide-for-medium, .hide-for-large – hide for specific viewports
  • .show-for-small, .show-for-medium, .show-for-large – explicitly show on specified viewports

Typography

All elements, typography included, are reset by normalize.css and therefore absolutely bare-bones. Styling is up to you.

  • .text-left – align text to left
  • .text-center – center text
  • .text-right – align to right

Effects

  • .tfx – short for "transition effects"; applies 0.3s linear animation to transitions (used on buttons)
  • .zoom – scale element to 110% by default

Helper Classes

  • .clearfix – *any comments are futile*
