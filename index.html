BRICKst framework

BRICKst is yet another mobile-first CSS framework for kickstarting your projects. Inspired by ZURB's Foundation (I mean, I pretty much copied their grid solution), OOCSS, SASS amazing powers and more.

Grid

12-column rows by default, nestable (one level confirmed :)

  • .row – classic row with max-width defined
  • .row.narrow – row with smaller gutters between columns (by half)
  • .row.collapse – no gutters between columns at all
  • .row.expand – full-width row

Buttons

5 sizes, 5 variants (default, secondary, success, alert, warning)
